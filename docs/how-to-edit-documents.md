# 如何编辑飞书文档 - 完整指南

## 问题概述

通过 `Rong_AiAgent` 创建的文档，你可能无法直接编辑。这是正常的，因为文档的所有者是飞书应用，需要先设置权限。

## 🔧 解决方案（二选一）

### 方案一：手动设置权限（推荐，快速）

这是最简单的方法，不需要配置任何飞书应用权限。

**步骤：**

1. **打开文档**
   - 点击机器人发送给你的文档链接

2. **点击「分享」**
   - 在文档右上角找到「分享」按钮并点击

3. **添加你自己为协作者**
   - 点击「邀请协作者」
   - 搜索你的名字
   - 选择权限：**「可编辑」**
   - 点击「确定」

4. **完成！**
   - 现在你可以编辑文档了

---

### 方案二：配置飞书应用权限（一劳永逸）

如果你想让 `Rong_AiAgent` 自动设置权限，需要在飞书开放平台配置权限。

**步骤：**

#### 1. 打开飞书开放平台
访问：https://open.feishu.cn/app/cli_a9636a165179dbc2/auth

#### 2. 添加权限
点击「权限管理」，搜索并添加以下权限之一：

**推荐权限：**
- `docs:permission.member:create` - 添加云文档协作者
- `docs:doc` - 查看、评论、编辑和管理文档

**其他可选权限：**
- `drive:drive`
- `drive:file`
- `sheets:spreadsheet`
- `wiki:wiki`
- `bitable:bitable`
- `bitable:app`

#### 3. 申请发布新版本
1. 进入「版本管理与发布」
2. 点击「创建版本」
3. 填写版本信息
4. 点击「申请发布」

#### 4. 等待审核
发布审核通过后，`Rong_AiAgent` 就能自动设置权限了！

---

## 📋 对比

| 方案 | 优点 | 缺点 |
|------|------|------|
| 手动设置 | 快速，无需配置 | 每次创建文档都需要手动设置 |
| 配置应用权限 | 自动化，一劳永逸 | 需要申请权限和发布 |

---

## 💡 当前 `Rong_AiAgent` 的工作流程

```
1. ✅ 创建文档
2. ⚠️ 尝试自动设置权限（失败但不影响）
3. ✅ 发送文档链接给你
4. 📝 你在飞书中手动设置权限（可选但推荐）
```

---

## 🔍 为什么需要这样？

- 飞书应用创建的文档，所有者是应用本身
- 要让其他用户访问，需要显式设置权限
- 飞书API需要特定的应用权限才能通过代码设置权限
- 这是飞书的安全机制，防止未授权的权限操作
